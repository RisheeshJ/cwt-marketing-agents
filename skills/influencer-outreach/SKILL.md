# Influencer Outreach Skill

## When triggered
Use for tasks about finding influencers, building creator databases, or
drafting outreach in the retail-trading niche.

## Procedure
1. Using web search and the built-in browser tools, find 8-12 REAL influencers
   in retail trading with >200K subs/followers on YouTube, X, Instagram or
   TikTok. Verify counts on their actual channel/profile pages where possible;
   mark unverified numbers as "approx". Never invent data.
2. Capture everything useful per influencer: name, handles, platforms,
   follower counts, content style, topics, audience, posting frequency,
   engagement signals, contact method, fit rationale.
3. Write `Influencers/Influencer-Database.md` in the vault: one master table
   plus a short profile per influencer.
4. Write `Outreach/Cold-Outreach-Drafts.md`: for each influencer a
   personalized message (max 120 words) that references a specific piece of
   their content, introduces crowdwisdomtrading.com in one sentence, and asks
   for their honest OPINION about it (soft-touch, not a paid-promo pitch).
   Adapt tone per platform (email vs X DM vs IG DM).
5. As a kanban worker, kanban_complete with a summary and file paths.
